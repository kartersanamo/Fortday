# Mixed OG Baseline Ruleset

## Core Assumptions
- Mechanics-first over perfect visual parity in early milestones.
- Mixed Chapter 1-era weapon and movement feel.
- Competitive loop before cosmetics or progression systems.

## Match Loop
1. Lobby party and queue
2. Waiting island countdown
3. Match start and drop phase
4. Loot/build/fight/storm loop
5. Victory and post-match summary

## Timing and Fidelity Guards
- Building placement pipeline target <= 100ms server-confirm under stable 50ms ping.
- Editing confirm/reset cadence tuned against OG reference footage.
- Storm phases use deterministic seed for repeatable balancing tests.
